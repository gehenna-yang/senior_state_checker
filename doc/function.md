# 앱 기능 상세 정의
## 사용자 등록 기능
- 사용자 계정 Id, Password 를 입력받아 생성

## 로그인 기능
- 앱 실행시 계정이 로그인되어있지 않는경우 로그인 후 기능 사용 가능

## 보호자 초대기능
- 사용자가 다른사용자에게 보호자 등록 요청을 하는경우를 말함
- 보호자 등록 요청을 받은 사용자가 승인해야 보호자로 등록됨

## 사용자 상태체크 항목을 생성, 수정, 삭제 기능
- 사용자 또는 보호자가 투약 항목을 등록가능
- 투약 항목 등록 또는 수정시 약에 대한 내용들을 입력 할 수 있음

```
예시) 혈압약, 당뇨약, 감기약 등등
```

## 보이스 인식을 통한 간편 상태 체크 기능 (STT 활용)
- 사용자가 보이스를 통해 투약, 운동, 식사 등을 기록 할 수 있도록 함.
- 미리 등록해둔 상태체크 항목과 보이스를 통한 내용이 동일한 경우 상태체크 항목으로 입력.

```
예시) "혈압약 먹었어", "오늘 아침 식사 완료", "운동 시작", "운동 종료" 등
```

## 운동 트레킹 기능
- 사용자가 '운동'을 기록하는경우 시작시간과 종료시간을 기록하고 움직임을 트레킹한다
- 움직임을 트레킹하는경우 km를 계산하여 기록

## 사용자 상태 기록
- 투약, 식사, 운동 등 설정된 항목에 대한 수행 여부를 기록
- 항목에 대한 기록은 보이스를 통해서 하거나 항목을 선택하여 기록 가능
- 기록시 시간은 자동으로 입력한다.
- 각 항목별 특이사항(메모) 입력 기능
- 사진 첨부를 통한 식사 또는 투약 증빙 기능

## 일별/월별 상태 기록 조회 기능
- 캘린더 뷰를 통해 일자별 상태 기록 여부를 시각적으로 표시
- 특정 날짜 선택 시 해당 일의 투약, 식사, 운동 기록 및 메모, 사진 상세 조회

## 보호자 알림 및 리포트 공유 기능
- 주간/월간 단위로 사용자의 상태 기록 리포트를 생성하여 보호자에게 공유
- 긴급 상황 발생 시 보호자에게 즉시 알림 기능

```
긴급상황은 설정을 통해 조건을 설정 할 수 있음
예시) 일정 시간 동안 활동이 감지되지 않거나, 필수 투약 항목이 누락된 경우 등
```

## 통계 및 분석 기능
- 주간/월간 통계를 통해 항목별 달성률(이행률) 제공
- 이전 기간 대비 상태 변화 추이 그래프 제공
- 보호자가 사용자의 건강 상태 변화를 한눈에 파악할 수 있는 대시보드 구성

## 설정 및 기타 기능
- 푸시 알림 설정 (투약 시간 알림, 식사 시간 알림 등)
- 보호자 연결 관리 (연결된 보호자 목록 확인 및 해제)
- 개인정보 수정 및 프로필 이미지 설정
- 로그아웃 기능
